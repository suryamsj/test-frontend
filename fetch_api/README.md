### Teknologi
Teknologi yang saya gunakan :
- Bootstrap 5
- Axios
- Svelte
- Routify

### Menjalankan Project
```bash
# Clone this project
$ git clone https://github.com/suryamsj/test-frontend

# Access
$ cd test-frontend/fetch_api

# Install dependencies
$ npm install

# Run the project
$ npm run dev

# The server will initialize in the <http://localhost:5000>
```
